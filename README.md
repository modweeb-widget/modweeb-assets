# Modweeb Assets 🎨

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

مجموعة من الأكواد والتصاميم الجاهزة لمدونات بلوجر.

## 📁 الهيكل

```
modweeb-assets/
├── assets/         # موارد إضافية (صور، خطوط)
├── css/            # أنماط CSS
├── js/             # سكريبتات JavaScript
└── svg/            # أيقونات SVG
```

## 🚀 الاستخدام

### استيراد الأنماط
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/modweeb-widget/modweeb-assets@latest/css/style.css">
```

### استيراد السكريبتات
```javascript
import AuthManager from 'https://cdn.jsdelivr.net/gh/modweeb-widget/modweeb-assets@latest/js/auth.js';
```
